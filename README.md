
# livekolkata# Kolkata 3D Live Digital Twin 🌆

An interactive 3D digital twin of Kolkata featuring a dual-engine architecture:
1. **WebGL Engine (Three.js):** Lightweight, real-time client with animated taxis, buses, metro, plane flight paths, weather effects, and street lighting.
2. **Unreal Engine 5 Mode:** WebRTC Pixel Streaming client for high-end photorealistic rendering with Cesium 3D Tiles.

## Features
- **Landmark Teleportation:** Smooth fly-to controls for Howrah Bridge, Victoria Memorial, Biswa Bangla Gate, Park Street, and Howrah Station.
- **Interactive Inspection:** Click on landmarks to inspect historical details.
- **Dynamic Weather & Lighting:** Toggle between Day, Night, and Monsoon Rain modes.
